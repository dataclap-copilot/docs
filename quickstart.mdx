# What is Kubetrace?

Kubetrace is a cloud-agnostic Kubernetes testing platform that provides:
- **Security validation** - RBAC, pod security, secrets, and policies
- **Performance testing** - Pod scaling, scheduling, and resource contention  
- **Network testing** - Connectivity, DNS, and network policies under stress
- **Storage validation** - Persistent volumes, dynamic provisioning, and failover
- **API testing** - Stress test Kubernetes API server under load
- **AI/ML optimization** - GPU usage and distributed training resilience

---

## Getting Started

### Step 1: Sign Up
1. Visit [app.kubetrace.net/register](https://app.kubetrace.net/register)
2. Create your free account (no credit card required)
3. Access the Kubetrace dashboard

### Step 2: Connect Your Cluster
1. **Select your cloud provider** in the Kubetrace platform
2. **Choose your cluster** from the available options
3. **Run the onboarding script** to create read-only service account

```bash
# Example onboarding command
./kubetrace-azure.sh my-aks-cluster readonly-kubeconfig.yaml
```

### Step 3: Deploy the Operator
1. **Upload your kubeconfig** to the platform
2. **Generate the operator YAML** from the dashboard
3. **Deploy the operator** to your cluster

```bash
kubectl apply -f kubetrace-operator.yaml
```

### Step 4: Start Testing
- **Run automated test suites** across Security, Performance, Network, Storage, and API
- **Schedule recurring tests** (hourly, daily, weekly, monthly)
- **View real-time results** and topology visualization
- **Get compliance reports** and risk assessments
