# Troubleshooting

Find solutions to common problems encountered while using Kubetrace. If your issue is not listed, please contact support at [support@kubetrace.co](mailto:support@kubetrace.co).

## Common Issues

### 1. Cannot Connect Cluster
- Ensure your kubeconfig is correct and up to date.
- Check network connectivity between your machine and the Kubernetes API server.

### 2. Operator Deployment Fails
- Verify you have the necessary permissions to deploy resources.
- Check for typos or missing fields in the YAML file.

### 3. Test Results Not Showing
- Make sure the operator is running and healthy in your cluster.
- Refresh the dashboard and check logs for errors.
